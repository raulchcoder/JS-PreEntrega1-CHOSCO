Registro de Ingresos y Gastos

Sistema para registrar los INGRESOS y GASTOS, utilizando Javascript. Para INICIAR presionar el botón EJECUTAR, para DETENER el sistema presione la botón CANCELAR.

Para visualizar el REPORTE de lo registrado es necesario habilitar la consola del navegador.

Pasos a seguir

  1. Seleccionar tipo de registro: 1 - Ingreso, 2 - Gasto.
  2. Ingresar descripción.
  3. Ingresar monto o importe.
  4. Para terminar la carga de datos presione CANCELAR.
